Welcome! This interactive web application is designed to help students understand the most straightforward way computers try to represent negative numbers: Sign-and-Magnitude. 

When studying data representation in your Computer Systems topic, it is much easier to see the binary bits change on screen than trying to calculate everything mentally. This tool makes the concepts visual!

## 🎯 What You Will Learn
* **The Sign Bit in Action:** See exactly how the leftmost bit operates just like a plus (+) or minus (-) sign.
* **The "Negative Zero" Flaw:** Discover the `-0` bug by exploring the app. Why is having two different binary codes for the number zero a bad thing for computer engineering?
* **Why Complements Were Invented:** Through the "Common Misconception" section, learn why simply adding binary numbers with signs doesn't work, which will smoothly prepare you for learning One's Complement and Two's Complement next.

## 🚀 How to Use This Tool
1.  **Open the File:** No installation is needed. Just double-click the `index.html` file to open it in your web browser (Chrome, Safari, Edge, etc.).
2.  **Input Numbers:** Type a decimal number (between -127 and 127) directly into the large input box, or use the `+1` / `-1` buttons to change the value step-by-step.
3.  **Observe the Binary:** As the number changes, look at the 8 bits on the right. Notice which bits highlight and flip when the number goes from positive to negative.
4.  **Find the Bug:** Click the **"Show -0"** button to reveal the hidden computer design flaw that scientists eventually had to fix.
